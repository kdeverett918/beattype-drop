# BeatType Drop

A single-file rhythm typing arcade game. Words fall on the beat, and the player clears each word by typing it before it reaches the drop line.

## Play Locally

Double-click `index.html`, or serve the folder with any static file server.

## Deployment

This repo is configured for Render as a static site:

- Build command: `true`
- Publish directory: `.`

The game itself has no external runtime dependencies.
